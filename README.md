# FlowMail Campaigns – Email Campaigns for WordPress

FlowPress is a simple email automation plugin for WordPress. Create subscriber emails, manage campaigns, queue messages, and upgrade to WooCommerce automations when you need store-based workflows.

No SaaS. No monthly fees. Runs inside WordPress.

## Download

Download the latest free version here:

https://github.com/thamilton9883/Flowpress/releases/latest

## Free core features

- Subscriber welcome automations
- Birthday and anniversary automations
- Email campaigns
- Built-in subscribe forms
- Email queue management
- SMTP support
- Form styling settings

## WooCommerce upgrade

FlowPress Commerce adds store-focused automations:

- Abandoned cart recovery
- Post-purchase follow-up sequences
- Review request emails
- Repeat customer automations
- High-value customer workflows
- Revenue tracking

Learn more:

https://flowpress.uk/

## Installation

1. Download the plugin zip from the latest release.
2. In WordPress, go to Plugins → Add New → Upload Plugin.
3. Upload the zip file.
4. Activate FlowPress.
5. Open FlowPress from the WordPress admin menu.
